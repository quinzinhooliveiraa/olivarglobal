Como configurar um Google Ads MCC para sua agência

O MCC (Manager Account) é a conta-mãe do Google Ads que permite gerenciar várias contas de clientes de um único lugar. Sem ele, você precisa fazer login em cada conta separadamente, o que vira um caos rápido.

Por que o MCC é indispensável

Com o MCC você consegue:

  • Ver performance de todos os clientes numa tela só
  • Fazer ajustes em múltiplas contas sem trocar de login
  • Ter acesso de agência sem depender das credenciais do cliente
  • Criar relatórios consolidados
  • Gerenciar usuários e permissões centralizadamente

Como criar o MCC passo a passo

  1. Acesse ads.google.com/intl/en_us/home/tools/manager-accounts/
  2. Clique em "Create a Manager Account"
  3. Use um e-mail que não esteja vinculado a nenhuma conta de Google Ads existente. Recomendamos criar um e-mail específico para a agência (ex: ads@suaagencia.com)
  4. Preencha o nome da conta (nome da sua agência) e selecione o país e fuso corretos. Para clientes americanos, selecione United States e o fuso da maioria dos seus clientes.
  5. Confirme a criação. Pronto — você tem um MCC.

Como adicionar clientes ao MCC

  • Opção A — Cliente existente te dá acesso: o cliente acessa a conta dele no Google Ads → Ferramentas → Acesso e segurança → Gerentes → Solicitar link → Insere o ID do seu MCC.
  • Opção B — Você cria a conta para o cliente: dentro do MCC, clique em "+ New Google Ads account" → cria a conta já vinculada ao MCC. O cliente nunca precisa mexer na configuração.

Opção B é a mais recomendada para clientes novos. Você tem controle total desde o início e não depende do cliente para configurações técnicas.

Estrutura de contas recomendada

Cada cliente deve ter:

  • Uma conta própria no Google Ads (nunca misture clientes na mesma conta)
  • Faturamento configurado no cartão do próprio cliente — não no seu. Isso evita problemas se o cliente cancelar ou se houver disputa de cobrança.
  • Acesso de administrador para você e acesso de leitura para o cliente (se ele quiser acompanhar)

Permissões: o que dar para o cliente

Recomendamos dar ao cliente acesso de Read-only (somente leitura). Isso permite que ele veja tudo, impressões, cliques, gastos, sem poder pausar campanhas acidentalmente ou fazer alterações não autorizadas.

Se o cliente pedir acesso de admin, explique que alterações sem coordenação podem prejudicar o desempenho das campanhas. A maioria entende.

Ponto importante: faturamento

Nunca coloque o cartão da agência como método de pagamento padrão dos clientes. Caso o cliente cancele e você esqueça de pausar as campanhas, o gasto vem para você.

O fluxo correto: cliente coloca o cartão dele → campanhas rodam com o dinheiro dele → você gerencia o uso. Para clientes que preferem que você gerencie o budget, use uma conta de faturamento separada por cliente ou trabalhe com crédito pré-pago.
