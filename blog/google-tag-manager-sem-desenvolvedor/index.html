Como usar Google Tag Manager sem precisar de desenvolvedor

O Google Tag Manager (GTM) é a ferramenta que permite instalar e gerenciar códigos de rastreamento em sites sem precisar mexer no código fonte. Para agências, isso significa independência total do desenvolvedor para configurar conversões, eventos e pixels.

Os três conceitos do GTM

  • Tags: os códigos que você quer executar no site. Pode ser o código de conversão do Google Ads, o pixel do Facebook, o Google Analytics, CallRail, etc.
  • Triggers (gatilhos): as condições que ativam as tags. "Quando o formulário for submetido", "quando o usuário clicar no botão de ligação", "quando acessar a página /thank-you".
  • Variables (variáveis): valores dinâmicos usados em tags e triggers. O mais comum é a URL da página atual.

Instalação no site: 2 minutos

  1. Acesse tagmanager.google.com → crie uma conta para o cliente
  2. Copie os dois snippets de código fornecidos pelo GTM
  3. Cole o primeiro snippet no <head> do site (o mais alto possível)
  4. Cole o segundo snippet logo após a tag <body>
  5. Publique o container do GTM

Para sites em WordPress, use o plugin "Site Kit by Google" que instala o GTM automaticamente. Para sites em Wix, Webflow ou Squarespace, existe campo nativo para inserir scripts no header.

Configurando rastreamento de conversão do Google Ads

  1. No Google Ads, crie a conversão: Ferramentas → Conversões → + → Website
  2. Selecione a ação (ex: "Book dumpster"), configure o valor e copie o Conversion ID e Conversion Label
  3. No GTM, crie uma nova Tag → Google Ads Conversion Tracking → insira o ID e Label copiados
  4. Configure o Trigger: selecione "Page View" e filtre para URL contém "/thank-you" (ou a URL da página de obrigado)
  5. Nomeie a tag, salve e publique

Em 5 minutos, conversões de formulário estão sendo rastreadas sem tocar no código.

Rastreando cliques no botão de telefone

Esse é o rastreamento mais importante para home services, muitos leads ligam diretamente sem preencher formulário.

  1. No GTM, crie um Trigger → Tipo: "Click" → configure para links que começam com "tel:"
  2. Crie uma Tag de conversão do Google Ads apontando para esse trigger
  3. Publique

Agora cada clique no número de telefone no site é registrado como conversão no Google Ads.

Preview mode: como testar antes de publicar

Antes de qualquer publicação no GTM, ative o Preview Mode (botão no canto superior direito). Isso abre uma janela de debug que mostra em tempo real quais tags estão disparando em quais eventos.

Navegue pelo site do cliente em modo preview, simule as ações (preencha o formulário, clique no número) e verifique se as tags corretas aparecem como "Fired".

Só publique depois de confirmar que tudo está funcionando no preview.

Versionamento: nunca perca o que funcionava

O GTM mantém histórico de todas as versões publicadas. Se você fizer uma mudança que quebra algo, basta ir em Versões e publicar a versão anterior.

Nunca faça mudanças no GTM de produção sem testar no preview primeiro, é uma das boas práticas mais importantes.
