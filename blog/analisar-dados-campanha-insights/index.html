Como analisar os dados de uma campanha e gerar insights para o cliente

Ver número é fácil. Transformar número em decisão é o que diferencia uma agência que gera resultado de uma que gera relatório.

Esse guia cobre as análises mais importantes para campanhas de home services e o que fazer com cada insight.

Search Terms Report: o relatório mais importante

O Search Terms Report (Termos de pesquisa) mostra exatamente o que as pessoas digitaram antes de clicar no seu anúncio. É onde você descobre:

  • Quais termos estão gerando conversões (potencialmente novas keywords a adicionar)
  • Quais termos estão gerando cliques irrelevantes (negativar imediatamente)
  • Como acessar: Palavras-chave → Termos de pesquisa → Filtrar por período.

Ação: revise semanalmente nos primeiros 30 dias de campanha, quinzenalmente depois. Adicione negativos agressivamente para qualquer termo que não seja intenção de compra clara.

Exemplos de termos para negativar em dumpster rental:

  • "dumpster rental jobs" (alguém procurando emprego)
  • "how to build a dumpster" (pesquisa informacional)
  • "dumpster rental \cidade fora da área de serviço\"

Análise de Quality Score

O Quality Score (QS) determina quanto você paga por clique. QS alto = CPC menor com mesma posição.

O QS tem três componentes:

  • Expected CTR: o anúncio é relevante para a busca?
  • Ad Relevance: a mensagem do anúncio corresponde à intenção de busca?
  • Landing Page Experience: a página que o usuário encontra é relevante e rápida?
  • QS abaixo de 5 é sinal vermelho. Acima de 7 é bom. 9-10 é excelente.
  • Ação: para keywords com QS baixo, verifique qual dos três componentes está fraco e ajuste o anúncio ou a landing page correspondente.

Análise de Impression Share

Impression Share mostra qual porcentagem das buscas elegíveis o anúncio apareceu. Se IS = 60%, em 40% das buscas você não apareceu.

Existem dois motivos para IS baixo:

  • Budget insuficiente: campanha fica sem verba durante o dia
  • Baixo Ad Rank: concorrentes têm QS ou lance maior

Ação: se "Impression Share Lost to Budget" for alto, o cliente precisa aumentar o orçamento ou você precisa limitar o horário de exibição para concentrar o budget nos horários de maior conversão.

Análise de performance por hora do dia

Como acessar: Campanhas → Segmentos → Hora do dia.

Essa análise revela quando seus clientes estão comprando. Para home services americanos, o padrão mais comum é pico entre 10h e 14h nos dias de semana.

Ação: configure ajuste de lance por horário — aumente em 20-30% nos horários de pico, diminua em 20-30% nos horários de baixa performance. Se o orçamento for limitado, pause completamente fora dos horários de conversão.

Análise de performance por dispositivo

A maioria dos leads de home services vem de mobile. Se o custo por conversão no desktop é 3x maior que no mobile, pode valer pausar desktop ou reduzir o lance significativamente.

Ação: aplique ajuste de lance negativo para dispositivos com CPA muito acima da média. Nunca pause completamente sem dados suficientes (mínimo 50 cliques por dispositivo).

Como transformar análise em comunicação para o cliente

Não envie dump de dados — envie insight acionável:

!❌ "CTR aumentou de 3.2% para 4.1%"

!✅ "Os anúncios com foco em 'same-day delivery' tiveram CTR 28% maior que os outros. Vou expandir essa mensagem para todas as campanhas."

!❌ "Gastamos $800 na semana"

!✅ "$800 investidos geraram 23 leads. Destes, 8 confirmaram como clientes, gerando $2.400 em receita. ROI de 3x."
